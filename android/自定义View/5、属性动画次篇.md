# 一、TypeEvaluator
## 1.ArgbEvaluator 来做颜色渐变的动画
**例子1：**
> 使用系统自带的ArgbEvaluator
```java
ObjectAnimator animator = ObjectAnimator.ofInt(view, "color", 0xffff0000, 0xff00ff00);  
animator.setEvaluator(new ArgbEvaluator());  
animator.start();  
```
