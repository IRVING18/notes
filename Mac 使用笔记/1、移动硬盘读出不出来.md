今天Mac装一个软件之后直接把移动硬盘直接拔下来了，然后想再插上弄点的东西的时候发现不识别了，瞬间一头冷汗，以为移动硬盘坏掉了，然后赶紧Google发现了使用命令行挂载硬盘就行。

## 第一步先查看硬盘的信息使用如下命令行

```java
 diskutil list
```

输出如下

## 第二步 发现/dev/disk2是我的移动硬盘然后使用挂载命令直接挂载就行如下

```java
sudo  fsck_hfs -fy /dev/disk2
在执行装载命令之前先执行上边这行命令
sudo diskutil mount /dev/disk2
```
