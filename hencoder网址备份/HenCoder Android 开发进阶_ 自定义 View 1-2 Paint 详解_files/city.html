@BJ173986 也就是srcBitmap的透明区域一定要能包含dstBitmap。
如果是自己绘制的bitmap，可以在大小上做一些处理：
//只是在400*400的区域中间绘制了一个半径100的圆形，剩余的透明空间足够包含dst
val src = Bitmap.createBitmap(400, 400, Bitmap.Config.ARGB_8888)
Canvas(src).apply {
val paint = Paint()
paint.color = Color.RED
drawCircle(200.0f, 200.0f, 100.0f, paint)
}

val dst = Bitmap.createBitmap(200, 200, Bitmap.Config.ARGB_8888)
Canvas(dst).apply {
val paint = Paint()
paint.color = Color.GREEN
drawRect(0.0f, 0.0f, 200.0f, 200.0f, paint)
}

val saveCount = canvas.saveLayer(null, null, Canvas.ALL_SAVE_FLAG)
canvas.drawBitmap(dst, 200.0f, 200.0f, mPaint)
mPaint.xfermode = PorterDuffXfermode(PorterDuff.Mode.DST_IN)
canvas.drawBitmap(src, 200.0f, 200.0f, mPaint)
mPaint.xfermode = null
canvas.restoreToCount(saveCount)
